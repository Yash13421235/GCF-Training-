JavaScript Functions Masterclass

Learn Every Important Function Concept in JavaScript with Interactive Examples

Basic Function

1. Normal Function

A normal function is created using the function keyword.

function greet(name){
  return "Hello " + name;
}
Arrow Function

2. Arrow Function

Arrow functions are shorter and modern syntax for functions.

const add = (a,b) => a+b;
Anonymous

3. Anonymous Function

A function without a name is called anonymous function.

let x = function(){
  return "Anonymous";
}
Callback

4. Callback Function

A callback is passed into another function.

function process(callback){
  callback();
}
Default Parameter

5. Default Parameters

Default values are used if no value is passed.

function user(name="Guest"){
 return name;
}
Rest Parameter

6. Rest Parameters

Rest parameter collects multiple values into array.

function total(...nums){
 return nums;
}
Return

7. Return Statement

Return sends data back from function.

function square(x){
 return x*x;
}
Recursive

8. Recursive Function

A function calling itself is recursive function.

function factorial(n){
 if(n===1) return 1;
 return n*factorial(n-1);
}
IIFE

9. Immediately Invoked Function

Runs instantly after creation.

(function(){
 console.log("Run")
})();
Higher Order

10. Higher Order Function

Functions that take other functions as arguments.

arr.map(x => x*2)
Closure

11. Closure

Closure remembers variables from outer function.

function outer(){
 let count = 0;
 return function(){
   count++;
 }
}
Async

12. Async Function

Async functions work with promises.

async function data(){
 return "Done";
}
