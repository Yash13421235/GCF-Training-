About Math Object

The Math Object in JavaScript provides mathematical constants and functions. It allows developers to perform calculations like rounding numbers, generating random numbers, finding powers, square roots, trigonometric values, and much more.


Main Features

  • Math.round() → Round to nearest integer
  • Math.floor() → Round down
  • Math.ceil() → Round up
  • Math.random() → Generate random numbers
  • Math.max() → Largest number
  • Math.min() → Smallest number
  • Math.pow() → Power calculation
  • Math.sqrt() → Square root
  • Math.abs() → Absolute value
  • Math.sin(), Math.cos(), Math.tan() → Trigonometry
  • Math.PI → Value of π

Math.round()

Rounds a number to the nearest integer.

Math.round(4.6)

Math.floor()

Always rounds DOWN.

Math.floor(8.9)

Math.ceil()

Always rounds UP.

Math.ceil(2.1)

Math.random()

Generates a random number between 0 and 1.

Math.random()

Math.max()

Returns the largest value.

Math.max(10, 40, 99, 5)

Math.min()

Returns the smallest value.

Math.min(10, 40, 99, 5)

Math.pow()

Finds power of a number.

Math.pow(2, 4)

Math.sqrt()

Finds square root.

Math.sqrt(81)

Math.abs()

Converts negative numbers into positive.

Math.abs(-55)

Math.PI

Returns value of PI.

Math.PI

Math.sin()

Returns sine value.

Math.sin(90 * Math.PI / 180)

Math.cos()

Returns cosine value.

Math.cos(0)

Math.tan()

Returns tangent value.

Math.tan(45 * Math.PI / 180)

Math.trunc()

Removes decimal part.

Math.trunc(7.99)

Math.sign()

Checks if number is positive or negative.

Math.sign(-20)

Important Notes

  • The Math object does not need new keyword.
  • Math object is static.
  • You directly use methods like Math.sqrt().
  • Most trigonometry functions use radians, not degrees.
  • Math.random() is commonly used in games.
