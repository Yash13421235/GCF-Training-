JavaScript Arrays Complete Guide

Learn JavaScript Arrays with examples, explanations, and important methods.

1. What is an Array?

An array is a special variable that can store multiple values in a single variable. 

// Creating an array
let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits);

2. Accessing Array Elements

Array values are accessed using index numbers. Index starts from 0. 

let colors = ["Red", "Green", "Blue"];

console.log(colors[0]); // Red
console.log(colors[1]); // Green

3. Array Length

let numbers = [10, 20, 30, 40];

console.log(numbers.length);

The length property returns the total number of elements.

4. push() Method

push() adds a new element at the end of the array. 

let fruits = ["Apple", "Banana"];

fruits.push("Orange");

console.log(fruits);

5. pop() Method

pop() removes the last element from the array. 

let fruits = ["Apple", "Banana", "Mango"];

fruits.pop();

console.log(fruits);

6. shift() Method

let numbers = [1, 2, 3];

numbers.shift();

console.log(numbers);

shift() removes the first element.

7. unshift() Method

let numbers = [2, 3, 4];

numbers.unshift(1);

console.log(numbers);

unshift() adds element at the beginning.

8. Loop Through Array

let fruits = ["Apple", "Banana", "Mango"];

for(let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

Loop is used to access every element one by one.

9. forEach() Method

let numbers = [10, 20, 30];

numbers.forEach(function(num){
    console.log(num);
});

forEach() runs a function for each array item.

10. map() Method

let numbers = [1, 2, 3];

let doubled = numbers.map(function(num){
    return num * 2;
});

console.log(doubled);

map() creates a new array by transforming elements.

11. filter() Method

let numbers = [1, 2, 3, 4, 5];

let even = numbers.filter(function(num){
    return num % 2 === 0;
});

console.log(even);

filter() creates a new array with matching values.

12. find() Method

` let result = numbers.find(function(num){ return num > 10; }); console.log(result);

find() returns the first matching value.

13. includes() Method

let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits.includes("Banana"));

includes() checks if a value exists.

14. sort() Method

let numbers = [4, 2, 1, 3];

numbers.sort();

console.log(numbers);

sort() arranges elements in order.

15. reverse() Method

let numbers = [1, 2, 3, 4];

numbers.reverse();

console.log(numbers);

reverse() reverses the array order.

16. concat() Method

let arr1 = [1, 2];
let arr2 = [3, 4];

let result = arr1.concat(arr2);

console.log(result);

concat() joins arrays together.

17. slice() Method

let numbers = [1, 2, 3, 4, 5];

console.log(numbers.slice(1,4));

slice() returns selected elements.

18. splice() Method

let numbers = [1, 2, 3, 4];

numbers.splice(1,2);

console.log(numbers);

splice() adds or removes elements.

19. Array Destructuring

let colors = ["Red", "Green", "Blue"];

let [a, b, c] = colors;

console.log(a);

Destructuring stores array values into variables.

20. Multidimensional Arrays

let matrix = [
    [1, 2],
    [3, 4]
];

console.log(matrix[0][1]);

Arrays inside arrays are called multidimensional arrays.
